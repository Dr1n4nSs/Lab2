# Задание 1
## Задача 10
### Текст задачи
Получить список из произведений цифр соответствующих чисел, хранящихся в
исходном списке.
### Алгоритм решения
Программа просит пользователя ввести колличество элементов в списке и сами элементы. Ввод проверяется с помощью Int32.TryParse:
Если введено не число Int32, то выдаётся ошибка.
Если введено число, то вызываются соответствующие функции и выводиться ответ
Первая функция получает элемент и делает его произведением его цифр при помощи остатка от деления на 10. Впоследствии эта функция применяется к каждому элементу при помощи List.map
Вторая функция считывает элементы, которые вводит пользователь в список при помощи цикла.

# Задание 2
## Задача 10
### Текст задачи
Список содержит строки, обозначающие двоичные числа от 1 до 9. Найти в
десятичной системе счисления сумму элементов списка
### Алгоритм решения
Программа просит пользователя ввести колличество элементов в списке и сами элементы. Ввод проверяется с помощью Int32.TryParse:
Если введено не число Int32, то выдаётся ошибка.
Если введено число, то вызываются соответствующие функции, далее, при помощи List.fold, Элементы складываются в ответ и он выводиться.
Первая функция принимает строку и проверяет, что эта строка является двоичным числом от 1 до 9 и возвращает это число, если оно подходит под этот критерий.
Вторая функция Используется для считывания введённых элементов в список через цикл.
