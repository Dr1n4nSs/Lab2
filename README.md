# Задание 1
## Задача 10
### Текст задачи
Сформировать список из модулей вводимых чисел.
### Алгоритм решения
Программа просит пользователя ввести, сколько чисел он планирует добавить. Ввод проверяется с помощью Int32.TryParse:
Если введено положительное число (n > 0), начинается цикл заполнения.
Если введено 0, выводится сообщение о пустом списке.
Если введен текст или некорректный символ, выводится ошибка.
Далее идёт заполнение списка. Внутри цикла for от 1 до n:
Программа запрашивает очередное число.
Используется Double.TryParse, чтобы разрешить ввод дробных чисел.
Число переводится в модуль.
Это число добавляется в список спереди.
Далее список разворачивается и выводится.

# Задание 2
## Задача 10
### Текст задачи
Вычислить факториал числа.
### Алгоритм решения
Программа считывает строку и пытается преобразовать её в целое число при помощи Int32.TryParse.
Если введено неотрицательное число (n больше или равно 0), вызывается функция вычисления.
Если введено отрицательное число, выводится предупреждение (так как факториал определён для натуральных чисел и нуля).
В случае ввода текста, выводится ошибка некорректного ввода.
Рекурсивная функция factorial:
Базовый случай: если n меньше или равно 1, функция возвращает 1. 
Рекурсивный шаг: если n больше 1, функция умножает текущее число на результат вызова самой себя с аргументом (n - 1).
В конце выводится результат.

# Задание 3
## Задача 2
### Текст задачи
Создайте собственные функции для выполнения основных операций над списками (добавление/
удаление/поиск элемента, сцепка двух списков, получение элемента по номеру).
### Алгоритм решения
Функции:
Добавление (addElement): Использует оператор :: для мгновенной вставки элемента в «голову» списка.
Сцепка (connectLists): Объединяет два списка в один новый с помощью оператора @.
Поиск (searchList): Перебирает список циклом for, сравнивая каждый элемент с искомым. Возвращает индекс первого совпадения, упакованный в тип Option (Some или None).
Удаление (removeElementByValue): Сначала ищет индекс элемента, а затем циклом собирает новый список, просто пропуская (игнорируя) элемент на этом индексе.
Доступ по индексу (getElement): Проверяет корректность индекса (чтобы он не был меньше нуля или больше длины списка) и с помощью цикла идёт до нужной позиции.
Далее реализовано меню, в котором происходит выбор операции или завершение программы 1-6, при выборе действия происходит вызов соответствующей функции.
